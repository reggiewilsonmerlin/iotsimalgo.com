2. What IOTSIM Is

IOTSIM (Infinite Outcome Trade Simulator + Insight Monitor — including the ICT Edition, "IOTSIMICT") is a research, analysis, and self-configured monitoring tool published as a TradingView indicator. It allows the user to define their own technical criteria — including but not limited to fair value gaps, order blocks, liquidity levels, volume imbalances, session windows, risk parameters, and entry/exit logic — and then:

  • simulates hypothetical trade outcomes against historical price data based on those user-defined configurations; and
  • monitors live price action on the user's chosen chart and notifies the user, via TradingView's native alert system, when conditions matching the user's own configuration occur.
